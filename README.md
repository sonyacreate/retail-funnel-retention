# Воронка и Retention: Online Retail II

## Описание проекта
Когортный анализ покупателей британского интернет-магазина —
исследуем как часто покупатели возвращаются и где бизнес теряет клиентов.

## Бизнес-контекст
Компания хочет увеличить возвращаемость покупателей.
Нужно понять на каком этапе клиенты уходят и есть ли сезонные паттерны в поведении.

## Данные
- Источник: [Kaggle — Online Retail II](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)
- Период: 2009–2011
- Исходный размер: 1 067 371 строк
- После очистки: 805 549 строк

## Что сделали
- Удалили 243 007 строк без Customer ID — без него невозможно отследить повторные покупки
- Удалили ~19 000 отменённых заказов (Invoice начинается на 'C')
- Построили воронку лояльности по количеству заказов на покупателя
- Построили когортную retention-матрицу по месяцам
- SQL-анализ через DuckDB: топ покупателей по выручке, новые клиенты по месяцам, доля стран в выручке с window functions

## Ключевые выводы
- 27.6% покупателей не возвращаются после первой покупки
- Только 35.3% делают заказ в следующий месяц после первой покупки
- Через 12 месяцев retention вырастает до 49.5% — покупатели возвращаются к НГ-сезону
- Великобритания даёт 82.98% выручки, но Ирландия, Нидерланды и Германия показывают потенциал роста
- Пик новых покупателей — декабрь 2009 и октябрь-ноябрь 2010, оба раза перед НГ

## Рекомендации для бизнеса
- Запустить email-цепочку после первой покупки — удержать новых клиентов
- За 1-2 месяца до НГ реактивировать покупателей прошлого года
- Инвестировать в экспансию на европейские рынки — особенно Ирландия и Нидерланды
- Разработать программу лояльности для стимулирования повторных покупок
